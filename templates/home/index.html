What is ITCH?

Visual Block Coding

ITCH uses a text-block style of programming. Instead of typing lines of code, you simply drag and drop function blocks to build your program logic.

Intuitive Logic

Read the different function blocks to understand their behavior, then snap them together. It works just like building with LEGO bricks!

Auto-Programming

Simply select the pins and the outputs you want. The system handles the heavy lifting and automatically generates the code for your board.
